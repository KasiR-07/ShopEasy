# Shop Easy - E-Commerce Web Application

## Overview

Shop Easy is a responsive e-commerce web application developed using HTML, CSS, and JavaScript. The project provides users with a simple and user-friendly shopping experience through essential pages such as Home, Products, Cart, and Contact.

This project demonstrates core front-end web development concepts, including responsive web design, DOM manipulation, interactive product displays, and shopping cart functionality.

---

## Features

### User Features

* Home page with project introduction and featured content
* Products page displaying available products
* Add products to the shopping cart
* View and manage items in the cart
* Contact page for user inquiries
* Responsive design for desktop, tablet, and mobile devices
* Interactive user interface using JavaScript

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/shop-easy.git
```

2. Navigate to the project directory

```bash
cd shop-easy
```

3. Open the project folder in your code editor.

4. Run the application by opening the `index.html` file in your browser.

Alternatively, use the VS Code Live Server extension for a better development experience.
